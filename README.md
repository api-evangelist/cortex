# Cortex (cortex)

Cortex is an Engineering Operations (EngOps) platform and internal developer portal that helps engineering teams catalog services, enforce production readiness with scorecards, automate self-service workflows, and surface engineering intelligence across their organization.

## APIs

### Cortex REST API

REST API for the service catalog, scorecards, initiatives, entity relationships and dependencies, on-call assignments, custom data, and deployments managed in a Cortex workspace.

- [Documentation](https://docs.cortex.io/)
- [API Reference](https://docs.cortex.io/reference)
- [Getting Started](https://docs.cortex.io/docs/getting-started)

### Cortex MCP Server

Model Context Protocol server that lets AI coding assistants query the catalog, ownership, scorecard scores, and run workflows from IDE agents.

- [Documentation](https://docs.cortex.io/)
